Open the web dashboard backed by your live behavior store.

Serves a local, read-only dashboard that visualizes the versions you have
captured with 'dow commit': the version tree, per-version configuration and
outputs, stability and custom evaluator metrics, and drift/verdicts between
versions (computed with the same engine as 'dow compare').

The dashboard reflects your .dow store live. Capture more versions with
'dow commit' in another terminal and refresh the page to see them. It binds to
localhost only; use --host/--port to change where it listens. Use --export to
write the underlying data as JSON without starting a server. Aliased as 'dow dash'.

@examples
dow dashboard
dow dashboard --spec summarization
dow dashboard --port 8080 --no-open
dow dashboard --export dow-data.json
